# SaaS Metrics

keywords: saas, 구독, subscription, churn, 이탈, retention, 리텐션, arpu, mrr

## Benchmarks

| metric | value | source |
|--------|-------|--------|
| 월간 이탈률 (Good) | 3-5% | ProfitWell 2024 |
| NDR (Net Dollar Retention) | 110-130% | Bessemer Cloud Index |
| CAC Payback Period | 12-18개월 | OpenView SaaS Benchmarks |
| LTV/CAC Ratio | 3:1 이상 | SaaS Capital |

## Terminology

| term | definition |
|------|-----------|
| NDR | 순 달러 유지율 — 기존 고객 매출이 확장(업셀) 포함해서 얼마나 유지되는지 |
| MRR | 월간 반복 매출 |
| ARPU | 사용자당 평균 매출 |
| Cohort Retention | 동일 가입 시기 고객군의 시간별 잔존율 |

## Common Decisions

- 이탈 방지에 리소스를 얼마나 투입할지
- 가격 정책을 변경할지
- 어떤 고객 세그먼트에 집중할지
