## Terminology

| term | definition |
|------|-----------|
| Agent Performance | 담당자 성과. 처리량·해결율·만족도 등을 기반으로 개인별 효율을 평가 |
| Performance Variance | 성과 편차. 상위 그룹과 하위 그룹 간 지표 차이. 클수록 교육/프로세스 문제 시사 |
| Coaching Opportunity | 코칭 기회. 하위 담당자의 성과를 상위 담당자 수준으로 올릴 수 있는 여지 |
| Throughput | 처리량. 단위 시간당 처리 건수 |
| Quality Score | 품질 점수. 처리 건의 정확도·만족도를 종합한 점수 |

## 분석 프레임워크

### 1. 상위/하위 그룹 비교
- 에스컬레이션율(또는 핵심 KPI) 기준으로 상위 5명 vs 하위 5명 분리
- 비교 축: 처리량, 해결율, 에스컬레이션율, 만족도, 평균 처리시간
- **카테고리 보정**: 같은 유형의 건을 처리해도 차이가 나는지 확인 (카테고리 배정 편향 제거)

### 2. 편차 해석 기준
| 편차 | 해석 | 액션 |
|------|------|------|
| 5%p 이내 | 정상 범위 — 개인차 | 유지, 모니터링 |
| 5-10%p | 교육 여지 존재 | 하위 그룹 대상 OJT, 해결 사례 공유 |
| 10%p+ | 구조적 교육 문제 | 체계적 교육 프로그램, 프로세스 표준화 |

### 3. 교육 ROI 추정
- 하위 그룹이 상위 그룹 수준으로 개선 시 전체 지표 개선 폭 계산
- 예: 하위 5명 에스컬레이션율 34%→28%(상위 수준)이면 전체 에스컬레이션 월 N건 감소

## 시각화 가이드

- 상위/하위/전체 비교 → DataTable (컬럼: 상위 평균, 하위 평균, 전체 평균, 편차)
- 개인별 분포 → HorizontalBarChart (담당자별 핵심 KPI 순위)
- 편차가 10%p+ 이면 InsightCard에서 "역량" 차원 원인으로 진단

## Common Decisions

| decision | required_info |
|----------|--------------|
| 교육/코칭 대상 선정 | 상담원별 핵심 KPI 순위, 카테고리 보정 후 편차, 하위 그룹 공통 패턴 |
| 인력 배치 최적화 | 상담원별 카테고리 적합도, 처리량 대비 품질, 과부하 임계선 |
| 성과 평가 기준 재설계 | 현재 평가 지표의 유효성, 처리량 vs 품질 가중치, 팀 vs 개인 기여도 |
